Mengapa PraktIoT?

Belajar IoT dengan PraktIoT Starter Pack 1.0

Pelatihan intensif yang menggabungkan teori dan praktik langsung. Sembilan proyek pertama berfokus pada dasar-dasar ESP32 dan kontrol nirkabel, diakhiri dengan proyek puncak membangun Smart Home Hub yang terhubung ke internet.

Pemrograman ESP32

Pelajari dasar-dasar pemrograman ESP32 dari nol hingga mahir. Mulai dari variabel, fungsi, hingga konektivitas WiFi dan kontrol sensor dengan pendekatan yang mudah dipahami.

Proyek Unggulan

10 Modul Proyek Interaktif PraktIoT 1.0

Setiap proyek dirancang untuk memberikan pengalaman praktis dalam mengembangkan solusi IoT menggunakan ESP32 untuk masalah nyata

01

Sistem Alarm Kebakaran

Bangun sistem deteksi dini kebakaran menggunakan ESP32 dan flame sensor dengan respons otomatis.

  • Deteksi api dengan flame sensor
  • Aktivasi alarm buzzer otomatis
  • Indikator LED peringatan
  • Monitoring lokal dengan ESP32
ESP32 Flame Sensor Buzzer
02

Pendeteksi Kebocoran Gas

Sistem monitoring gas berbahaya dengan ESP32 dan sensor gas, dengan peringatan dini dan simulasi ventilasi otomatis.

  • Deteksi gas LPG dan gas berbahaya
  • Sistem peringatan dini dengan buzzer
  • Simulasi ventilasi otomatis
  • Monitoring lokal dengan ESP32
ESP32 Gas Sensor Buzzer
03

Dot Matrix Berjalan

Ciptakan display teks berjalan dinamis menggunakan ESP32 dan dot matrix 8x8.

  • Kontrol dot matrix 8x8 via ESP32
  • Pemrograman teks berjalan
  • Penggunaan IC 74HC595
  • Multiple animation patterns
ESP32 Dot Matrix 74HC595
04

Pengukur Suhu Digital

Monitor suhu lingkungan secara real-time menggunakan ESP32 dengan display LCD.

  • Sensor suhu presisi dengan ESP32
  • Display hasil di LCD 1602
  • Peringatan suhu ekstrim dengan LED
  • Kalibrasi sensor suhu
ESP32 Temperature Sensor LCD 1602
05

Simulasi Lampu Lalu Lintas

Implementasi sistem lampu lalu lintas dengan ESP32, kontrol waktu otomatis dan mode darurat.

  • Kontrol LED RGB dengan ESP32
  • Logika timer yang dapat diprogram
  • Mode darurat dengan tombol
  • Simulasi persimpangan jalan
ESP32 LED RGB Timer
06

Lampu Otomatis dengan LDR

Sistem pencahayaan otomatis menggunakan ESP32 yang responsif terhadap kondisi cahaya.

  • Sensor cahaya LDR dengan ESP32
  • Kontrol lampu otomatis
  • Pengaturan sensitivitas cahaya
  • Mode manual dengan tombol
ESP32 LDR Sensor Relay
07

Tong Sampah Otomatis

Tong sampah pintar dengan ESP32 dan sensor jarak untuk membuka tutup otomatis.

  • Sensor ultrasonik dengan ESP32
  • Kontrol servo untuk membuka tutup
  • Pengaturan delay otomatis
  • Indikator LED status
ESP32 Ultrasonic Servo
08

Kontrol Servo dengan Remote

Sistem kontrol servo motor nirkabel menggunakan ESP32 dan remote IR.

  • Pemrograman IR receiver dengan ESP32
  • Kontrol servo SG90 dengan remote
  • Multi-angle positioning
  • Mode preset posisi
ESP32 Servo Motor IR Remote
09

Kontrol LED dengan Remote

Sistem pencahayaan LED interaktif dengan ESP32 untuk berbagai mode dan efek pencahayaan yang dapat dikontrol dengan remote IR.

  • Kontrol LED RGB dengan ESP32
  • Berbagai mode pencahayaan
  • Kontrol intensitas cahaya
  • Efek transisi warna
ESP32 RGB LED IR Remote
10

ESP32 Smart Home Hub

Proyek puncak yang mengintegrasikan semua komponen dengan ESP32 untuk menciptakan sistem rumah pintar lengkap dengan dashboard web yang terhubung ke internet.

  • Integrasi multi-sensor dengan ESP32
  • Kontrol rumah pintar terpusat
  • Dashboard web real-time
  • Konektivitas aplikasi seluler
ESP32 Integrasi IoT Smart Home
